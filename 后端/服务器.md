 * 场景1： 
   
    ![案例1](./imgs/403case1.png)
	
	原因1：    
	```
	1.此处是因为接口未开放，且请求种未携带token导致被security拒绝
	```
    解决方案：
    ```
    方式1：请求携带token等认证信息。
    方式2：开放该接口，让其无需认证。
	```
